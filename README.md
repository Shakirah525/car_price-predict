## 📊 Notebooks Overview

| # | Notebook | Description |
|---|----------|-------------|
| 1 | Data Gathering | Load dataset, inspect shape/dtypes, check missing values, analyze price distribution |
| 2 | Data Cleaning | Fix typos (maxda→mazda), handle missing values, remove duplicates, treat outliers |
| 3 | EDA | Univariate, bivariate, multivariate analysis — correlation heatmaps, pair plots |
| 4 | Feature Engineering | Create brand segment, power-to-weight ratio, engine categories, log transforms |
| 5 | Preprocessing Pipeline | Train/test split, OneHotEncoding, StandardScaler, ColumnTransformer pipeline |
| 6 | Model Training | Linear Regression + Ridge/Lasso, Random Forest, XGBoost with hyperparameter tuning |
| 7 | Model Evaluation | MAE/RMSE/R²/MAPE comparison, residual plots, SHAP values, business insights |

## 🎯 Success Criteria

- **R² Score > 0.80** — model explains 80%+ of price variance
- **Low MAE/RMSE** — small average prediction error
- **Business Insights** — actionable recommendations for dealerships

## 📦 Dataset

- **Source:** CarPrice_Assignment.csv
- **Rows:** 205 cars
- **Features:** 26 columns (price, engine specs, dimensions, brand, fuel type, etc.)

## 🔑 Key Findings

- Engine size, horsepower, and curb weight are the strongest predictors of car price
- Luxury brands (Jaguar, Porsche, BMW) command 2–3× the price of economy brands
- Rear-wheel-drive cars are generally more expensive than front-wheel-drive
- Price is right-skewed; log transformation improves model performance
